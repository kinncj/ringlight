# Ring Light
Control the elgato ring light

## Setup
### Dependencies
`pip install eglight`

`pip install click`

### Download
curl https://raw.githubusercontent.com/kinncj/ringlight/main/elgato > ~/.elgato

## Commands
- brightness
  - Sets the brightness
  - `~/.elgato brightness 12`
- temperature
  - Sets the temperature
  - `~/.elgato temperature 6900`
- toggle
  - Toggle ON/OFF
  - `~/.elgato toggle`
